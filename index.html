P3509 [POI 2010] ZAB-Frog

发表于 2025-10-06 分类于 学习

题目链接

P3509 [POI 2010] ZAB-Frog - 洛谷

题目大意

给出n个位置，对于每个位置的下一次会跳跃到距离该位置最近的第k个位置，有相同距离的选左边，询问对于每个位置跳跃m次之后会处在哪个位置。

解题思路

观察范围:1 ≤ k < n ≤ 106, 1 ≤ m ≤ 1018

首先就是想到了对于m次跳跃可以先处理出一次的然后根据倍增处理出跳跃m位之后的情况

那么对于处理出一次，可以使用单调队列来维护。

并且相比于窗口滑动这类题目更加简单，只需要维护两端的指针就好了。

那么对于每一个节点来说，只有两端的位置可能是下一个，显然远端（或是相同距离的左端）会成为下一个跳跃到的节点。

于是代码就写好了

信心满满一波提交！但是！喜提26分

记录详情 - 洛谷 | 计算机科学教育新生态

为什么呢，观察到内存只有125MB并且非常严格，直接导致空间的溢出造成WA和MLE.

于是博主百思不得起解查看了题解。

然后震惊于题解思路，非常巧妙地借鉴了快速幂的思想，使用一样的时间复杂度和优秀的空间复杂度完成了任务。

于是最终版本就是这样

发表于 2025-10-05 更新于 2025-10-06 分类于 学习

二叉搜索树Binary Search Tree

定义

是一棵满足中序遍历下节点的权值严格递增的特殊的二叉树.

那么实际上也就是符合下面两个条件：

  • 左子树的节点小于当前节点

  • 右子树的节点大于当前节点

  • 不存在重复节点

性质

  • 每个点的左右子树（如果存在的话）都是BST

  • 当前BST的最小节点一定是最左下的节点

  • 当前BST的最大节点一定是最右下的节点
